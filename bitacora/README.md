# Bitácora del Proyecto — ICIV Venezuela

Carpeta para registrar hallazgos, decisiones metodológicas, observaciones y notas de análisis
a lo largo de toda la investigación. Cada archivo es una entrada cronológica numerada.

## Convención de nombres

```
NN_etapa_tema.md
```

- `NN` — número secuencial (01, 02, 03…)
- `etapa` — fase del proyecto (EDA, limpieza, modelado, validacion, tesis…)
- `tema` — descripción breve del contenido

## Entradas

| # | Archivo | Etapa | Contenido |
|---|---|---|---|
| 01 | `01_EDA_hallazgos_iniciales.md` | EDA | Cobertura de datos, hallazgos cuantitativos clave, períodos críticos, decisiones metodológicas sugeridas |
| 02 | `02_decisiones_normalizacion.md` | Modelado | Log₁₀ tipo_cambio y deflactor, datos estáticos para gaps, transformaciones aplicadas en main.py |
